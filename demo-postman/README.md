## ubuntu安装

- 使用`snap`安装`postman`

  ```bash
  sudo snap install postman
  ```

- 启动`postman`

  ```bash
  postman
  ```


## Windows11安装

访问 https://www.postman.com/ 下载最新版本postman根据提示安装即可。
