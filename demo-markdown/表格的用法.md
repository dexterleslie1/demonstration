# 表格的用法

在Markdown中，你可以使用简单的语法来创建表格。以下是一个基本的Markdown表格语法示例：

```markdown
| 表头1 | 表头2 | 表头3 |  
| --- | --- | --- |  
| 单元格1 | 单元格2 | 单元格3 |  
| 单元格4 | 单元格5 | 单元格6 |  
| 单元格7 | 单元格8（可以很长） | 单元格9 |
```

这个Markdown代码会生成如下的表格：

| 表头1   | 表头2               | 表头3   |
| ------- | ------------------- | ------- |
| 单元格1 | 单元格2             | 单元格3 |
| 单元格4 | 单元格5             | 单元格6 |
| 单元格7 | 单元格8（可以很长） | 单元格9 |
