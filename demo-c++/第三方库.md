## fmt库是什么呢？

>说明：因为这个库是使用C++语言开发，所以C语言如果要调用这个库需要包装一层麻烦。

`fmt`（通常指 **{fmt}**）是一个用于 C/C++ 的**现代格式化库**，主要用来替代或增强 `printf / std::ostream`，特点是：**快、类型安全、接口简洁**。

---

### 1. fmt 是什么？

- 一个开源 C++ 格式化库
- 官网/源码：`github.com/fmtlib/fmt`
- 被 C++20 标准采纳（`std::format` 就是基于它）

一句话总结：  
**fmt 用来“把各种数据格式化成字符串”，比 printf 更安全，比 iostream 更快。**

---

### 2. 为什么不用 printf / cout？

#### printf 的问题
```c
printf("%d %f\n", a, b);
```
- 参数类型不对会直接 UB（未定义行为）
- 不支持自定义类型
- 可读性一般

#### iostream 的问题
```cpp
std::cout << a << " " << b << std::endl;
```
- 语法冗长
- 性能较差
- 格式化能力弱

---

### 3. fmt 的基本用法

#### 示例
```cpp
#include <fmt/core.h>

int main() {
    int x = 42;
    double y = 3.14;

    fmt::print("x={}, y={:.2f}\n", x, y);
}
```

输出：
```
x=42, y=3.14
```

特点：
- `{}` 自动推导类型
- 不会因类型错误崩溃
- 格式语法类似 Python

---

### 4. 常见格式写法（简表）

| 写法     | 含义            |
| -------- | --------------- |
| `{}`     | 默认格式        |
| `{:d}`   | 十进制整数      |
| `{:x}`   | 十六进制        |
| `{:.2f}` | 保留 2 位小数   |
| `{:<10}` | 左对齐，宽度 10 |
| `{:>10}` | 右对齐          |

---

### 5. 相比 printf 的优势

- 编译期检查格式字符串
- 支持自定义类型的 `formatter`
- 性能通常优于 iostream
- API 简单统一

---

### 6. 和 std::format 的关系

- C++20 的 `std::format` ≈ fmt 的子集
- fmt 功能更多、更新更快
- 老编译器也可以用 fmt

```cpp
// C++20
#include <format>
std::format("x={}", x);
```

---

### 7. 适合用在什么场景？

- 日志系统
- 高性能文本输出
- 需要类型安全的格式化
- 替代 printf / sprintf

### 8. 示例

详细用法请参考本站示例：https://gitee.com/dexterleslie/demonstration/blob/main/demo-c++/demo-fmtlib.cpp

安装fmt库

```sh
sudo apt update
sudo apt install libfmt-dev

# 检查是否安装成功
pkg-config --modversion fmt
```

demo-fmtlib.cpp

```c++
#include <fmt/core.h>

int main() {
    int a = 42;
    double b = 3.14159;

    fmt::print("a = {}\n", a);
    fmt::print("b = {:.2f}\n", b);
    fmt::print("Hello, {}!\n", "world");

    return 0;
}
```

编译

```sh
g++ demo-fmtlib.cpp -o main -lfmt
```

运行

```sh
./main
```
