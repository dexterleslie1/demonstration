## Puppeteer是什么呢？

Puppeteer 是一个由 Google 开发的 Node.js 库，它提供了一个高级 API 来控制 Chrome 或 Chromium 浏览器。通过 Puppeteer，你可以实现自动化操作，比如生成网页截图、抓取网页内容、模拟用户行为（如点击、输入、滚动等），以及进行前端性能测试等。它常用于爬虫、自动化测试、生成 PDF 等场景。
