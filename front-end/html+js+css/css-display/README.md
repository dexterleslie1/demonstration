# 演示css display用法

## display

### block

> 元素能够声明宽度和高度并且独占一行。
>
> div默认是块元素。
>
> 注意：在不声明元素宽度时，元素宽度会自动调整至其父级容器宽度，以填满父级容器宽度。

### inline

> 元素不能够声明宽度和高度并且不会独占一行。
>
> span默认是行内元素。

### inline-block

> 元素能够声明宽度和高度并且不会独占一行。

### table-cell

> 表格单元方式排列。
>
> 注意：能够声明宽度但是不能够声明高度，高度会自动被拉伸到适合单元格高度。

### none

> 元素被隐藏并且不会占据布局空间。visibility: hidden 元素隐藏并且会占据布局空间。
