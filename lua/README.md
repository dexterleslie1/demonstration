# lua脚本编程基础

## macOS安装lua环境

### 参考资料

> [Running Lua on macOS](https://rudamoura.com/luaonmacosx.html)  
> [下载对应macOS版本的lua pkg](http://rudix.org/)

### 安装和验证lua环境

```
# 双击运行lua-5.4.3-macos11.5.pkg安装lua环境

# 在终端命令行输入lua后，在lua shell中输入print("Hello world")，能够打印Hello world表示lua环境正确安装和配置了
```

## 使用lua命令运行lua脚本文件

```
# 运行hello-world.lua脚本文件
lua hello-world.lua
```
