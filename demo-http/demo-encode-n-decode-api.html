encodeURI / encodeURIComponent / decode 区别

规则：encodeURI 不编码 A-Z a-z 0-9 - _ . ! ~ * ' ( ) ; , / ? : @ & = + $ #
encodeURIComponent 只保留 A-Z a-z 0-9 - _ . ! ~ * ' ( )，其余都编码（常用于 query 参数值）

示例 1：含空格、中文、&、=

原串:

方法结果
encodeURI
encodeURIComponent

示例 2：完整 URL（encodeURI 保留 :// 等）

原串:

方法结果
encodeURI
encodeURIComponent

解码对应关系

decodeURI 对应 encodeURI；decodeURIComponent 对应 encodeURIComponent。
用错会报错或乱码，例如对 encodeURIComponent 的结果用 decodeURI 可能无法正确解码 %26 等。
