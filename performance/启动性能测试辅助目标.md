# 启动用于性能测试的辅助目标

## 启动`openresty`

- [下载用于性能测试的`openresty`](https://gitee.com/dexterleslie/demonstration/tree/master/performance/demo-perf-assistant-target-openresty)

- 编译`openresty docker`镜像

  ```bash
  docker compose build
  ```

- 运行`openresty`

  ```bash
  docker compose up -d
  ```

- 查看`openresty`服务是否正常，访问`http://localhost/`有输出表示服务正常
